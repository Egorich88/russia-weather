# 🌍 Russia Weather Monitoring 2026

## Полный DevOps проект мониторинга погоды по городам России

![Architecture](docs/architecture.png)

## 📋 О проекте
Система мониторинга погоды, развернутая на 3 виртуальных машинах с использованием современных DevOps инструментов. Собирает данные о температуре в городах России от Камчатки до Калининграда.

## ✅ Реализованные компоненты

| Компонент | Роль | Технологии |
|-----------|------|------------|
| **Jenkins VM** | CI/CD оркестратор | Jenkins, Docker, Ansible |
| **Prometheus VM** | Сбор и хранение метрик | Prometheus, Node Exporter |
| **Grafana VM** | Визуализация | Grafana, Prometheus Datasource |
| **Weather Exporter** | Генерация данных о погоде | Python, Prometheus Client, Docker |
| **Инфраструктура** | Управление VM | VirtualBox, Vagrant |
| **Автоматизация** | Настройка серверов | Ansible |

## 🚀 Быстрый старт

### Предварительные требования
- VirtualBox 6.1+
- Vagrant 2.3+
- Git
- 16GB+ RAM (рекомендуется)

### Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ваш-логин/russia-weather-2026.git
cd russia-weather-2026

# 2. Запустить все VM
vagrant up

# 3. Настроить Jenkins (первый вход)
# Пароль будет показан в консоли

# 4. Запустить pipeline в Jenkins
# deploy-monitoring → Build with Parameters → TARGET=all
